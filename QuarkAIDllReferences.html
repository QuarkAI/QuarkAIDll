    Supportive functions

    The supportive functions are used to support other functions, such as login QuarkAI CDKey, show messages, get HTTP text, document operations, etc.

    integer Login(string QuarkAICDKey)

  • This function is used to login your QuarkAI acount.
  • Attention: window controlling functions (with binding) will be disabled after
10 minutes without login. And you need to login your account before binding the 
window. Multiple login is not allowed, the new login will force the former login
to logoff. 
    •  Inputs:
      • QuarkAICDKey (string): QuarkAICDKey, such as "0123456789ABCDEF".
    •  Return value: 
Your QuarkAI ID as an integer. Each QuarkAI account has one email address,
one ID and lots of QuarkAI CDKeys.

    •  Example (VB Script):
      • 

      MyQuarkAIID = MyQuarkAI.Login("0123456789ABCDEF")

    Window information functions (without binding)

    The window information functions (without binding) are used to get the handle, process ID, window title, class name, process name, process path, etc. These functions can be used without binding the window.

    integer FindWindow(string ClassName, string WindowTitle)

  • Obtain the handle of a top window by its class name or its title.
  • Attention: ClassName and WindowTitle cannot be both empty.
    •  Inputs:
      • ClassName (string): the window class name, can be empty. Fuzzy matching.
WindowTitle (string): the window title, can be empty. Fuzzy matching.
    •  Return value: 
The handle of the window as an integer, 0 = no such a window.

    •  Example (VB Script):
      • 

      MyHandle1 = MyQuarkAI.FindWindow("ClassName", "Title")
MyHandle2 = MyQuarkAI.FindWindow("ClassName", "")
MyHandle3 = MyQuarkAI.FindWindow("", "Title")

    integer FindWindowEx(integer FatherWindowHandle, string ClassName, string WindowTitle)

  • Get the handle of a window by its father window, class name or title.
  • Attention: FatherWindowHandle is exact. Return of the first window's handle.

    •  Inputs:
      • FatherWindowHandle (integer): the father window handle, 0 = top window. 
ClassName (string): the window class name, can be empty. Fuzzy matching.
WindowTitle (string): the window title, can be empty. Fuzzy matching.
    •  Return value: 
The handle of the window as an integer, 0 = no such a window.

    •  Example (VB Script):
      • 

      MyHandle1 = MyQuarkAI.FindWindowEx(0 , "ClassName", "Title") 'Desktop window 
MyHandle2 = MyQuarkAI.FindWindowEx(FatherHandle, "Class", "") 'A child window
MyHandle3 = MyQuarkAI.FindWindowEx(FatherHandle, "", "Title") 'A child window

    • integer FindWindowByProcess(string ProcessName, string ClassName, string WindowTitle)

  • Get the handle of a window by its process name, class name or title.
  • Attention: ProcessName is exact. Return the handle of the first window found.
    • 

    •  Inputs:
      • ProcessName (string): the process name. Exact matching. 
ClassName (string): the window class name, can be empty. Fuzzy matching.
WindowTitle (string): the window title, can be empty. Fuzzy matching.
    •  Return value: 
The handle of the window as an integer, 0 = no such a window.

    •  Example (VB Script):
      • 

      MyHandle1 = MyQuarkAI.FindWindowByProcess("Process.exe", "Class", "Title") 
MyHandle2 = MyQuarkAI.FindWindowByProcess("Process.exe", "Class", "")
MyHandle3 = MyQuarkAI.FindWindowByProcess("Process.exe", "", "Title")

    string EnumWindow(integer ParentWindowHandle, string WindowTitle, string ClassName, integer Filter)

  • Get the handles of all matched windows by the parent window, class name or title.
  • Attention: ParentWindowHandle must be exact.

    •  Inputs:
      • ParentWindowHandle (integer): the parent window's handle. Exact matching. 
WindowTile (string): window title, which can be empty. Fuzzy matching.
ClassName (string): the class name, which can be empty. Fuzzy matching.
Filter (integer): searching policy. 1 = match title; 2 = match class name; 
4 = match the child windows; 8 = match top windows; 16 = match visible windows.
The value can be summed up for multiple polices such as 1+4+8.
    •  Return value: 
The handles of all matched windows as a string: "handle1,handle2,handle3".

    •  Example (VB Script):
      • 

      MyHandleList1 = MyQuarkAI.EnumWindow(0, "WindowTitle", "", 1+8+16) 
MyHandleList2 = MyQuarkAI.EnumWindow(FatherWindowHandle, "", "Button", 2+4)
MyHandleList3 = MyQuarkAI.EnumWindow(0,"Button", "", 1+8)

    • string EnumProcess(string ProcessName)

  • Get the process ID of all matched processes by matching process name.

    •  Inputs:
ProcessName (string): the process name. Fuzzy matching.
    •  Return value: 
The process IDs of all matched process as a string, like "PID1,PID2,PID3".

    •  Example (VB Script):
      • 

      MyPIDList = MyQuarkAI.EnumProcess("ProcessName.exe")

    • string EnumWindowByProcess(string ProcessName, string WindowTitle, string ClassName, integer Filter)

  • Get the handles of all matched windows.
  • Attention: ProcessName must be exact.

    •  Inputs:
      • ProcessName (string): the process name. Exact matching. 
WindowTile (string): window title, which can be empty. Fuzzy matching.
ClassName (string): the class name, which can be empty. Fuzzy matching.
Filter (integer): searching policy. 1 = match title; 2 = match class name; 
4 = match the first process; 8 = match top windows; 16 = match visible windows.
The value can be summed up for multiple polices such as 1+4+8.
    •  Return value: 
The handles of all matched windows as a string: "handle1,handle2,handle3".

    •  Example (VB Script):
